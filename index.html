About the Vault

A personal library catalog & notes system designed for students and avid readers — track books, annotate insights, and never lose a page reference again.

Features

  • 📖 Catalog with title, author, pages, tag, ISBN & notes
  • 🔍 Live regex-powered search with match highlighting
  • 📊 Dashboard: stats, reading goal, 7-day trend chart
  • ✏️ Inline table row editing
  • 💾 Auto-save + JSON & CSV import/export
  • 🌙 Dark / light theme
  • ♿ Full keyboard navigation & ARIA live regions
  • 📱 Mobile-first responsive design
